# LifeBalance-Manager

## Description
This project is an application designed to help users balance work, school, and personal hobbies by creating a realistic weekly schedule.

## Purpose
Many people, including myself, struggle to balance full-time work, online classes, and personal interests such as hobbies. This application aims to help people organize their time in a healty way without becoming overwhelmed.

## Features (Planned)
- Input work schedule
- Input school/study time
- Input hobbies (reading, hiking, cycling, etc.)
- Automatically generate a balanced weekly schedule
- Prioritize important responsibilities while including personal time

## Target Users
- Students with jobs
- Busy professionals
- Anyone trying to balance multiple responsibilities
